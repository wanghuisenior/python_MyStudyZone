Bootstrap Jquery-Confirm快速使用

依赖:

  • Bootstrap >= v2
  • jQuery library > v1.8

Alert

$.alert() 只有确认按钮. 

$.alert({
    title: '标题!',
    content: '简单提示!',
    confirm: function(){
        $.alert('确认了!'); // shorthand.
    }
});

Confirm

$.confirm() 包含确认和取消按钮. 

$.confirm({
    title: '确认吗!',
    content: '是否创建任务!',
    confirm: function(){
        $.alert('已确认!');
    },
    cancel: function(){
        $.alert('已取消!')
    }
});

跳转页面

HTML 
<a class="go" data-title="跳转?" href="http://www.baidu.com">跳转</a>
Javascript 
$('.example2-1-1').confirm(options); // 不需要写点击事件了

$('#id').confirm用来替代原本的confirm().
默认点击确认按钮执行跳转，若定义的`options`中包含回调函数，将取消跳转并执行回调
使用this.$target 获取到点击的元素.

Dialog

$.dialog() 同时隐藏确认和取消按钮. 显示 [x] 按钮 

$.dialog({
    title: 'Text content!',
    content: 'Simple modal!',
});

用法小结

一个简单的弹窗接收两个字符串参数： 第一个是对话窗的内容 第二个是窗口标题 .
第二个是可选参数. 

$.alert('Content here', 'Title here');
$.confirm('A message', 'Title is optional');
$.dialog('Just to let you know');

定制化

提示: $.confirm(), $.dialog() & $.alert() 三个方法都是 jconfirm()的别名.
三个方法都间接调用了 jconfirm 的基础函数

ADVANCED: this.$body is the body div for jquery-confirm. You can find and alter any element at run time.

窗口内元素隐藏

隐藏Title/Content/Confirm Button/Cancel Button/Close 按钮，如果你不需要他们.

$.confirm({
    title: false, // 隐藏标题.
    content: false, // 隐藏内容.
    cancelButton: false // 隐藏取消按钮.
    confirmButton: false, // 隐藏确认按钮.
    closeIcon: false, // 隐藏右上角关闭图表.
});
提示:

如果确认按钮和取消按钮同时被隐藏了，则默认情况下显示关闭图标（dialog mode）
若要明确的设定关闭图标的隐藏或显示，使用 closeIcon: true 或 closeIcon: false.
隐藏两个按钮的快捷方式是使用 $.dialog().

Ajax loading

使用jconfirm可以通过Ajax直接加载内容，而不需要额外的代码。.
有两种方法可以通过Ajax加载内容:

  1. 在字符串内容中传递准备好的URL
    如: content: "URL:http://example.com/getData?id=1"
  2. 返回jQuery Ajax承诺的传递函数.
    如: content: function(){ return $.get(...); }
在Ajax请求完成之前，确认和取消按钮都将变为不可用的状态.

使用 "URL:" 前缀

使用url前缀是最简单的方式，但是它有一些限制，比如你不能更改Ajax请求的参数 method,dataType, 等.
这种方式需要你准备好你的url，附带"URL:"参数,如 "URL:http://example.com/file.extension".
请求完成后 将会执行contentLoaded 回调函数，参数： Data, Status & Xhr object.

view text.txt 
$.confirm({
    content: 'url:/static/alert/jqueryConfirm/load_from_ajax.txt',
    title: 'Title'
});

在contentLoaded回调函数被调用之前已经设置好content内容 

$.confirm({
    content: 'url:/static/alert/jqueryConfirm/load_from_ajax.txt',
    title: 'Title',
    contentLoaded: function(data, status, xhr){
        var self = this;
        setTimeout(function(){
            self.setContent('<h1>OK! the status is: ' + status + '</h1><br>' + self.content);
            self.setTitle('title is redefined');
        }, 2000);
    }
});

使用 Ajax promise

此选项提供对Ajax选项的完全控制，以及将插入哪些数据 内容采用一个返回jQuery promise 的函数（$.ajax，$.GET，$.POST等）. 在这个例子中，请求一个JSON对象，并且其中的一部分被设置为内容。

view bower.json 
$.confirm({
    content: function () {
        var self = this;
        return $.ajax({
            url: 'bower.json',
            dataType: 'json',
            method: 'get'
        }).done(function (response) {
            self.setContent('Description: ' + response.description);
            self.setContent(self.content + '<br>Version: ' + response.version); // appending
            self.setTitle(response.name);
        }).fail(function(){
            self.setContent('Something went wrong.');
        });
    },
    confirm: function(){
        this.setContent( this.content + '<h4>Adding a new sentence.</h4>');
        return false; // prevent modal from closing
    }
});

自动关闭

如果用户在指定时间内未响应，则进行操作.
This comes in handly when the user is about to do something critical.
当用户将要做一些关键的事情时，这是恰好发生的。
这个autoClose 选项 接收一个字符串参数，如 'confirm|4000'表示4秒后自动执行某事件
自动关闭实例效果 


$.confirm({
    title: 'Logout?',
    content: 'Your time is out, you will be automatically logged out in 10 seconds.',
    autoClose: 'confirm|10000',
    confirm: function(){
        alert('confirmed');
    },
    cancel:function(){
        alert('canceled');
    }
});
$.confirm({
    title: 'Delete user?',
    content: 'This dialog will automatically trigger \'cancel\' in 6 seconds if you don\'t respond.',
    autoClose: 'cancel|6000',
    confirm: function(){
        alert('confirmed');
    },
    cancel:function(){
        alert('canceled');
    }
});

用户点击对话框外面的背景是否可以关闭对话框.

$.confirm({

捕获键盘动作

在jquery-confirm dialog中激活键盘事件.
ENTER按钮触发 confirm();事件 & ESC 触发 cancel();事件 

$.confirm({
    keyboardEnabled: true,
    content: 'Press ESC or ENTER to see it in action.',
    cancel: function(){
        $.alert('canceled');
    },
    confirm: function(){
        $.alert('confirmed');
    }
});

自定义键盘按钮

取人按钮和取消按钮接收数字键盘按键对应的数字, 如 65 代表 'A' 或 66 代表 'B'.

如果对话框中有文本输入框，在用户输入时将聚焦到输入框中，这时 ENTER & SPACE 按钮将不触发任何事件。
此时使用自定义键盘 确认 或 取消 按键，如 'A' 或 'B' 这样当你在对话框输入对应按键时仍然可以触发对话框事件，起到关闭或确认按钮的作用 
$.confirm({
    keyboardEnabled: true,
    content: 'Press "A" to confirm or "B" to cancel. <input type="text" placeholder="typing a or b will close the modal"/>',
    confirmKeys: [65],
    cancelKeys: [66],
    cancel: function () {
        $.alert('canceled');
    },
    confirm: function () {
        $.alert('confirmed');
    }
});

回调

对模态框进行更多的控制，主要用于模态框元素的绑定事件。.
contentLoaded 回调函数 is called when data from URL prefix in content is used. 

$.confirm({
    content: 'Imagine this is a complex form and you have to attach events all over the form or any element <br>' +
             '<button type="button" class="examplebutton">I\'ve events attached!</button>',
    onOpen: function(){
        alert('after the modal is opened/rendered');
        // find the input element and attach events to it.
        // NOTE: `this.$content` is the jquery object for content.
        this.$content.find('button.examplebutton').click(function(){
            alert('I\'ve powers!');
        });
    },
    onClose: function(){
        alert('before the modal is closed');
    },
    onAction: function(action){
        // action is either 'confirm', 'cancel' or 'close'
        alert(action + ' was clicked');
    }
});

onOpen()函数

这个函数在模态框打开前触发. 如果你使用URL前缀通过url加载内容，你需要在内容加载完毕后使用contentLoaded 回调函数

onClose()函数

无论通过何种方式关闭了模态框，这个函数都将被触发.

onAction()函数

当确认/取消按钮或者关闭图标被点击的时候会触发.这个onAction()函数传递一个字符串参数（被点击的按钮）.
